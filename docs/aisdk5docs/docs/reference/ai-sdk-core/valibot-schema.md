# [`valibotSchema()`](#valibotschema)






`valibotSchema` is a helper function that converts a Valibot schema into a JSON schema object that is compatible with the AI SDK. It takes a Valibot schema as input, and returns a typed schema.

You can use it to [generate structured data](../../ai-sdk-core/generating-structured-data.html) and in [tools](../../ai-sdk-core/tools-and-tool-calling.html).

## [Example](#example)



``` ts
import  from '@ai-sdk/valibot';import  from 'valibot';
const recipeSchema = valibotSchema(  object(),    ),    steps: array(string()),  }),);
```
